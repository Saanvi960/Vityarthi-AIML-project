# Marks Predictor using Linear Regression

##  Project Overview

This project is a simple Machine Learning model that predicts student marks based on the number of study hours. It demonstrates the use of basic AI concepts using Python.

##  Objective

The objective of this project is to build a predictive model that can estimate a student's marks by analyzing how many hours they study.

##  Technologies Used

* Python
* NumPy
* Scikit-learn

##  Machine Learning Algorithm

* Linear Regression (Supervised Learning)

##  Workflow

1. The model is trained on a dataset containing study hours and corresponding marks.
2. It learns the relationship between hours studied and marks obtained.
3. The user enters study hours as input.
4. The model predicts the expected marks.

## Setup

 Step 1: Install dependencies(scikit-learn numpy)

 Step 2: Run the project

 Sample Input & Output:-
  **Input:**
  Enter study hours: 5
  **Output:**
  Predicted Marks: 65

##  Features

* Simple and easy to understand
* No external dataset required
* Fast execution

##  Conclusion

This project shows how machine learning can be used to make predictions based on simple data. It helps in understanding the basics of AI and regression models.
